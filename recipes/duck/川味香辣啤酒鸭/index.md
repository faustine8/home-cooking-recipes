# 啤酒鸭菜谱

用量按 **2人份、嫩鸭半只800–1000克、普通炒锅带盖焖烧** 计算。文中的“大勺”约15毫升，“小勺”约5毫升。

# 一、买菜时看

* **主料：** 嫩鸭半只，800–1000克，让老板剁成3–4厘米块。
* **新鲜配料：** 姜1大块、蒜1头、大葱半根、青椒2个、红椒1个、蒜苗3根。
* **香料：** 干辣椒5–8个、花椒1小勺、八角2个、香叶2片、桂皮1小段。
* **调味料：** 郫县豆瓣酱、生抽、老抽、蚝油、料酒、冰糖、盐、啤酒1罐（约500毫升）。

最终口味是 **酱香微辣、鸭肉入味、酒香融合、青椒蒜苗清香提味**。
这道菜的重点不是“土豆炖鸭”，而是 **鸭肉本身的香味 + 啤酒焖出来的复合香气 + 最后青椒蒜苗的提鲜**。

# 二、备菜时看

## 1. 处理鸭肉

鸭块先用清水浸泡 **15–20分钟**，中间换一次水，泡掉明显血水。

如果鸭子看起来比较新鲜、腥味不重，**这道菜可以不焯水，直接煸炒**，这样鸭味更浓。

如果你担心腥味，也可以先简单焯一下：

锅中放鸭块，加足量冷水，再加 **姜4片、大葱几段、料酒1大勺**。

**大火煮开，撇浮沫；保持中大火再煮2分钟。** 捞出鸭块，用温水冲净，彻底沥干。

> 不管焯不焯水，后面都一定要把鸭块充分煸炒出油、炒干水汽，这一步最关键。

## 2. 切配菜

**青椒：** 切滚刀块。
**红椒：** 切块，或者用1–2根小米辣代替。
**蒜苗：** 切4厘米段，**蒜白和蒜叶分开装**。
**姜：** 切6–8片。
**蒜：** 8瓣，轻拍一下即可。
**大葱：** 切段。
**干辣椒：** 剪成两段。

## 3. 提前分好四份料

**香料碗：** 姜6–8片、蒜8瓣、大葱几段、干辣椒5–8个、花椒1小勺、八角2个、香叶2片、桂皮1小段，全部装进同一个碗。

**酱汁碗：** 生抽2大勺、老抽半大勺、蚝油1大勺，混合均匀。

**豆瓣小碟：** 郫县豆瓣酱1大勺。

**糖料小碟：** 冰糖5–6颗。

另外把 **啤酒1罐（约500毫升）** 提前打开，放在灶台旁边备用。

# 三、炒菜时手忙脚乱只看这里

## 1. 先把鸭肉煸干出油

炒锅烧热，放 **1小勺食用油**，倒入全部鸭块。

保持 **中大火**，不断翻炒 **8–10分钟**。

开始时鸭肉会出很多水，不要急着加调料，继续炒。炒到：

* 水汽基本散掉；
* 鸭皮开始出油；
* 鸭肉表面微微发黄；
* 锅底只剩比较清亮的鸭油。

如果锅里鸭油很多，可以舀出一部分，只留 **1–2大勺底油**。油太多会腻。

> 啤酒鸭香不香，关键就在这里。鸭肉没煸干，成品容易发腥发闷。

## 2. 加料酒

沿锅边淋入 **1大勺料酒**。

保持 **大火**，快速翻炒 **约30秒**，把明显酒气炒散。

料酒不用多，主要是去腥提香；后面还有啤酒，放多了酒味会杂。

## 3. 加香料碗

倒入准备好的 **香料碗**。

保持 **中火**，翻炒 **约1分钟**，把姜、蒜、葱和香料的味道炒出来。

## 4. 加豆瓣酱

把鸭肉和香料拨到锅边，锅中央放入 **豆瓣小碟里的郫县豆瓣酱**。

转 **小火**，炒 **30–45秒**，看到红油出来后，再和鸭肉混合翻炒。豆瓣酱不要大火久炒，容易糊、会发苦。

## 5. 加酱汁碗和冰糖

倒入准备好的 **酱汁碗**，再放入 **冰糖5–6颗**。

转 **中大火**，翻炒 **1–2分钟**，让酱汁均匀裹住鸭肉，把颜色和酱香炒出来。

## 6. 加啤酒焖烧

倒入 **啤酒约500毫升**，液体高度达到鸭肉的 **四分之三到基本持平** 即可，不需要完全淹没。

先开 **大火，不盖盖，煮3–5分钟**，把明显的生酒气挥发掉。

然后盖上锅盖，转 **小火**，保持轻微咕嘟，焖 **30–35分钟**。

中途翻动 **1–2次**，避免锅底糊住。

如果中途汤汁明显太少，可以补一点 **热水**，不要再补冷啤酒。

## 7. 开盖检查

时间到后开盖，先夹出 **八角、桂皮、香叶**。

夹一块鸭肉，用筷子能比较轻松扎进去，但肉块还完整，就是合适。

如果仍明显偏硬：继续盖盖，**小火再焖8–10分钟**。

## 8. 回锅收汁

确认鸭肉已经熟透后，转 **中大火收汁5–8分钟**。

收汁过程中隔十几秒翻动一下，避免底部糊锅。收到 **锅底还剩小半碗浓汁** 时，就可以加最后的配菜。

## 9. 加青椒、红椒、蒜苗

先下：

* 青椒块
* 红椒块
* 蒜苗白色部分

保持 **大火**，翻炒 **约1分钟**。

再下 **蒜苗叶**，继续翻炒 **20–30秒**，立刻关火。

最后尝味道：

* 淡：补少量盐，每次最多四分之一小勺；
* 颜色浅：补少量老抽，每次四分之一小勺；
* 太咸：加少量热水，再煮开；
* 辣味不足：此时可补一点小米辣，再炒半分钟。

收到 **汤汁能挂在鸭肉上、青椒断生、蒜苗还保持翠绿** 时关火。不要把汁完全收干，留一点拌饭最好吃。

## 最简操作口令

**中大火煸鸭出油 → 大火烹料酒 → 中火下香料碗 → 小火炒豆瓣 → 中大火倒酱汁和冰糖 → 倒啤酒，大火散酒气 → 小火焖30–35分钟 → 中大火收汁 → 大火下青红椒和蒜白 → 最后下蒜苗叶快速出锅。**
