# 蒜香微辣豆豉鲮鱼油麦菜

用量按 **2人份、油麦菜500克、甘竹牌豆豉鲮鱼半罐**计算。

最终口味是：**豆豉咸香、鲮鱼酥香、蒜味突出、微微带辣，油麦菜清脆不出水。**

# 一、买菜时看

* **主料：** 油麦菜500克、甘竹牌豆豉鲮鱼半罐。
* **新鲜配料：** 蒜5–6瓣。
* **微辣配料：** 干辣椒2个。
* **调味料：** 白糖少许。

这道菜一般**不需要盐、生抽、蚝油和豆瓣酱**。豆豉鲮鱼本身已经有足够的咸味、鲜味和油脂。

# 二、备菜时看

## 1. 处理油麦菜

油麦菜掰开，放入清水中彻底洗净，尤其注意根部和叶片缝隙里的泥沙。

洗好后充分沥干，最好再用厨房纸吸掉明显水珠。油麦菜表面水分太多，下锅后容易变成煮菜。

切成约 **6厘米长的段**，把比较粗的菜梗和叶片大致分开放：

* **菜梗部分：** 先下锅；
* **叶片部分：** 后下锅。

## 2. 处理豆豉鲮鱼

打开罐头，取出约 **半罐鲮鱼**，再取：

* 罐头里的豆豉约 **1大勺**；
* 罐头里的油约 **1.5大勺**。

鲮鱼用筷子或勺子分成 **2–3厘米的小块**，不要完全碾碎，否则炒出来只有碎末，没有鱼肉口感。

豆豉如果结成大块，稍微拨散即可。

## 3. 准备配料

* 蒜5–6瓣，拍碎后切成蒜末；
* 干辣椒2个，剪成小段；
* 白糖准备约 **三分之一小勺**。

把油麦菜、鲮鱼、豆豉、蒜末、干辣椒和白糖全部提前放在灶台旁边。这道菜全程很快，中途不要停下来找材料。

# 三、炒菜时手忙脚乱只看这里

## 1. 爆香蒜和辣椒

炒锅烧干，加入罐头里的油约 **1.5大勺**。

开 **中小火**，放入蒜末和干辣椒。

翻炒 **约15秒**，闻到蒜香、辣椒稍微变深即可。不要把蒜炒黄炒焦，否则会发苦。

## 2. 炒香豆豉

加入准备好的 **豆豉1大勺**。

保持 **中小火**，翻炒 **10–15秒**，让豆豉的香味进入油中。

豆豉不要久炒，炒太久容易发干、发苦。

## 3. 先下油麦菜梗

立即转 **最大火**。

倒入油麦菜比较粗的菜梗部分，快速翻炒 **约20–30秒**。

菜梗表面开始变绿、稍微变软时，马上进入下一步。

## 4. 再下油麦菜叶

倒入全部油麦菜叶。

保持 **最大火**，快速翻炒 **约40–60秒**。

刚下锅时菜很多，不要加水。不断从锅底向上翻，让每片菜叶都接触到热锅。

炒到油麦菜明显缩小、叶片刚刚变软，但菜梗仍然挺括时即可。不要等到大量出水。

## 5. 加入鲮鱼

倒入准备好的鲮鱼块，再撒入 **三分之一小勺白糖**。

保持 **大火**，快速翻炒 **20–30秒**。

动作要轻，主要是把鲮鱼和油麦菜混合均匀，不要持续铲碎鲮鱼。

白糖不是为了炒甜，而是稍微中和豆豉的咸味和油麦菜的微苦味。

## 6. 尝味出锅

夹一小段油麦菜尝味道：

* **咸味合适：** 立即关火出锅；
* **明显偏淡：** 最多补一小撮盐；
* **偏咸：** 不要加水，可以再补一把油麦菜快速炒熟。

正常使用甘竹牌豆豉鲮鱼时，通常不需要另外加盐。

从油麦菜第一次下锅到关火，尽量控制在 **2分钟以内**。炒好后立即装盘，不要留在热锅里继续焖，否则油麦菜会出水、变软、颜色发暗。

## 最简操作口令

**罐头油中小火爆蒜和干辣椒 → 下豆豉炒香 → 最大火先炒菜梗20秒 → 下菜叶炒至刚软 → 加鲮鱼和一点糖 → 大火翻炒20秒 → 不放盐直接出锅。**
