# 香辣猪脚菜谱

用量按 **2人份、猪脚700–900克、红皮小花生120克、燃气灶高压锅**计算。文中的“大勺”约15毫升，“小勺”约5毫升。

# 一、买菜时看

- **主料：** 前猪蹄1只，700–900克，让老板烧毛、剁成3–4厘米块；红皮小花生120克。
- **新鲜配料：** 姜1块、蒜1头、大葱1根。
- **香料：** 干辣椒6–8个、花椒1小勺、八角2个、香叶2片、桂皮1小段。
- **调味料：** 郫县豆瓣酱、生抽、老抽、蚝油、料酒、冰糖、盐。

最终口味是**酱香微辣、猪脚软糯、花生入味、浓汁挂肉**。干辣椒按你们耐辣程度调整，第一次不要超过8个。

# 二、备菜时看

## 1. 泡花生

红皮小花生 **120克** 洗净，冷水浸泡 **2–4小时**；泡好后沥水，红皮不用去。

## 2. 焯猪脚

猪脚洗净，尤其冲掉骨头切口处的碎骨和血水。锅中放猪脚，加足量冷水，再加 **姜4片、大葱半根、料酒1大勺**。

**大火煮开，撇浮沫；保持中大火再煮6分钟。** 捞出猪脚，用温水冲净，彻底沥干，避免炒糖色时溅油。

## 3. 提前分好四份料

**香料碗：** 姜6片、蒜6瓣、干辣椒6–8个、花椒1小勺、八角2个、香叶2片、桂皮1小段，全部装进同一个碗。

**酱汁碗：** 生抽3大勺、老抽半大勺、蚝油1大勺、料酒2大勺，混合均匀。

**豆瓣小碟：** 郫县豆瓣酱1大勺。

**糖色小碟：** 冰糖8-12颗。

另外烧一壶热水，准备约 **700–900毫升**。高压锅失水少，水不需要完全没过猪脚。

# 三、炒菜时手忙脚乱只看这里

## 1. 炒糖色

炒锅烧干，放 **1大勺食用油和糖色小碟里的冰糖**。

全程 **小火**，不断推动冰糖。冰糖完全融化，颜色变成琥珀色、出现密集小泡时，马上进入下一步。不要等到深褐色，否则会苦。

## 2. 立刻加入猪脚

**马上把全部猪脚倒入锅中。**注意防溅。

转 **中大火**，快速翻炒 **2–3分钟**，让每块猪脚都裹上糖色。

> 这一阶段不要停下来找调料：猪脚必须提前放在灶台旁边。

## 3. 加香料碗

倒入准备好的 **香料碗**。

保持 **中火**，翻炒 **约1分钟**，炒出姜、蒜和香料的香味。

## 4. 加豆瓣酱

把猪脚和香料拨到锅边，锅中央放入 **豆瓣小碟里的郫县豆瓣酱**。

转 **小火**，炒 **30–45秒**，看到红油出来后，再和猪脚混合翻炒。豆瓣酱不要大火久炒，容易糊。

## 5. 加酱汁碗

倒入准备好的 **酱汁碗**。

转 **中大火**，翻炒 **约1分钟**，让酱汁均匀裹住猪脚。

## 6. 转入高压锅

把炒锅里的猪脚、香料和全部酱汁倒入高压锅；加入泡好的 **红皮小花生120克**。

倒入热水，水面达到食材高度的 **约三分之二到四分之三**，不必完全淹没。翻动一下，防止花生全部压在锅底。

检查高压锅总容量，食材和汤水不要超过锅内安全刻度，通常不要超过锅体的三分之二。

## 7. 高压炖煮

盖好高压锅。

先开 **大火**，等高压锅上汽、压力阀稳定工作后，立即转 **小火**，保持稳定排气即可，不要让压力阀剧烈喷气。
从稳定上汽开始计时，压 **18分钟**。

时间到后关火，静置 **12–15分钟自然泄压**。不要立即强行放气，猪脚油脂和汤汁较多，强行泄压容易喷溅。

## 8. 开锅检查

确认完全泄压后开盖。夹一块猪脚，用筷子能比较轻松扎入皮和筋，但整块仍能夹起来，就是合适。

如果仍然明显偏硬：重新盖锅，大火上汽后转小火，再压 **5分钟**，继续自然泄压。

## 9. 回锅收汁

把高压锅里的猪脚、花生和汤汁全部倒回炒锅。八角、桂皮、香叶可以顺手夹出。

先开 **大火煮开**，然后转 **中大火收汁5–10分钟**。收汁过程中每隔十几秒从锅底轻轻推动，避免花生沉底粘锅，但不要猛烈翻炒，以免猪脚脱皮、花生碎掉。

汤汁明显变浓后尝味道：

* 淡：补少量盐，每次最多半小勺；
* 颜色浅：补少量老抽，每次半小勺；
* 太咸：加少量热水，再煮开；
* 辣味不足：此时可加切碎的小米辣，再煮1分钟。

收到 **汤汁能挂在猪脚上、锅底还剩小半碗浓汁**时关火。不要完全收干，花生关火后还会继续吸汁。

## 最简操作口令

**小火炒糖色 → 立刻下猪脚，中大火炒匀 → 中火下香料碗 → 小火炒豆瓣 → 中大火倒酱汁碗 → 转高压锅，加花生和热水 → 大火上汽、小火压22分钟 → 自然泄压 → 中大火收浓汁。**
