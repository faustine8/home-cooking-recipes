# 山药香菇乌鸡汤菜谱

用量按 **2人份、乌鸡半只约700–900克、干香菇6朵、铁棍山药350克、燃气灶高压锅** 计算。文中的“大勺”约15毫升，“小勺”约5毫升。

# 一、买菜时看

* **主料：** 新鲜乌鸡半只，约700–900克，让老板去鸡屁股、清理内腔、剁成4–5厘米的大块。
* **主要配菜：** 铁棍山药350克、干香菇6朵。
* **新鲜配料：** 生姜1块、小葱2根。
* **可选配料：** 红枣4颗、枸杞10克。
* **调味料：** 料酒、盐、白胡椒粉。

最终口味是 **鸡汤鲜香、香菇味柔和、山药粉糯、汤清但不寡淡**。不要放八角、桂皮、花椒、生抽或蚝油，这些味道会遮住鸡汤本身的鲜味。红枣最多放4颗，不喜欢汤带甜味可以不放。

# 二、备菜时看

## 1. 泡香菇

干香菇 **6朵** 快速冲洗掉表面灰尘，加入温水浸泡 **40–60分钟**。泡软后剪掉硬蒂，大朵切成两半，小朵保持完整。

泡香菇的水静置几分钟，取上层干净的 **150–200毫升** 留用，碗底带泥沙的部分倒掉。香菇水不要全部使用，否则汤色容易过深、香菇味过重。

## 2. 清理乌鸡

乌鸡块放入清水中仔细抓洗，重点洗掉鸡肚内部的血块、肺部残留、骨头切口处的碎骨和血水。黄色鸡油过多时去掉一部分，但不要全部去掉。

洗净后用清水浸泡 **15分钟**，中间换水一次，再捞出沥水。

## 3. 焯乌鸡

锅中放入乌鸡，加入足量冷水，再放 **姜4片、小葱1根打结、料酒1大勺**。

开 **大火**煮开，及时撇净浮沫；水开后保持 **中大火再煮3分钟**。捞出乌鸡，用温水冲净表面的浮沫，彻底沥水。

焯水锅里的汤全部倒掉，不要用来炖汤。

## 4. 处理山药

铁棍山药 **350克** 去皮，切成约4厘米长的滚刀块，放进清水中浸泡防止发黑。

处理山药时最好戴手套，避免山药黏液接触皮肤后发痒。山药暂时不要放进高压锅，和乌鸡一起压容易碎烂、使汤变得浑浊。

## 5. 提前分好三份料

**高压锅料：** 焯好的乌鸡、泡好的香菇、姜片5片、红枣4颗。

**液体：** 干净香菇水150–200毫升、热水约800–1000毫升、料酒1大勺。

**后放料：** 山药350克、枸杞10克、盐、白胡椒粉、葱花少许。

另外烧好一壶热水。高压锅失水少，汤水不需要加得太满。

# 三、炖汤时手忙脚乱只看这里

## 1. 食材放入高压锅

把 **焯好的乌鸡、香菇、姜片5片、红枣4颗** 放入高压锅，加入 **料酒1大勺和香菇水150–200毫升**。

再倒入热水，水面达到鸡块上方 **约2厘米** 即可，总液体量大约 **1000–1200毫升**。水不要放得过多，否则鸡汤味道会淡。

检查锅内总量，不要超过高压锅安全刻度，通常不要超过锅体容量的三分之二。

## 2. 高压炖乌鸡

盖好高压锅。

先开 **大火**，等高压锅稳定上汽、压力阀正常工作后，立即转 **小火**，保持压力阀稳定轻微排气即可，不要持续剧烈喷气。

从稳定上汽开始计时：

* 普通嫩乌鸡压 **12分钟**；
* 鸡块较大或乌鸡明显偏老，压 **15分钟**。

正常市场购买的半只乌鸡，先按 **12分钟** 处理。

时间到后关火，静置 **10–15分钟自然泄压**。不要立即强行放气，鸡汤较多，快速排气容易从排气阀喷出。

## 3. 开锅检查

确认压力完全释放后开盖。用筷子扎鸡腿或肉较厚的位置，能较轻松扎入、骨头附近没有血色，但鸡肉仍能保持完整，就是合适。

如果鸡肉仍明显偏硬，重新盖锅，大火上汽后转小火，再压 **3分钟**，继续自然泄压。

汤面油脂较多时，用汤勺撇掉一部分，只保留薄薄一层鸡油，不要全部撇净。

## 4. 加山药

把泡在水中的山药捞出，放入鸡汤。

此时 **不要再加压**，直接把高压锅当普通汤锅使用。开 **大火煮开**，然后转 **中小火**，虚掩锅盖煮 **10–15分钟**。

铁棍山药一般煮 **12分钟左右**。筷子能扎入、边角开始变圆润，但夹起来仍保持完整，就是合适。不要煮到完全碎进汤里。

## 5. 最后调味

山药煮熟后，加入 **盐1小勺、枸杞10克、白胡椒粉少许**。

保持 **小火煮2分钟**，然后尝味：

* 淡：每次补四分之一小勺盐；
* 汤太浓：补少量热水，再煮开；
* 香菇味太重：补100毫升左右热水，再煮2分钟；
* 油太多：关火静置2分钟，再撇掉表面多余鸡油。

不要加鸡精、生抽或蚝油。鸡汤刚开始尝起来不够鲜，通常是盐还没有放够，不要一次加太多盐。

## 6. 出锅

关火后撒少量葱花即可。喜欢汤味干净，可以不放葱花，只在碗中撒一点白胡椒粉。

山药会继续吸收汤汁，关火时汤要稍微多留一些，不要煮得过干。

## 最简操作口令

**香菇温水泡软 → 乌鸡加姜葱料酒冷水焯3分钟 → 乌鸡、香菇、姜片、香菇水和热水进高压锅 → 大火上汽、小火压12分钟 → 自然泄压 → 下山药，中小火煮12分钟 → 加盐、枸杞、白胡椒粉煮2分钟出锅。**
