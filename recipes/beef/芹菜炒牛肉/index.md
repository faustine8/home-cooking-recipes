# 芹菜炒牛肉菜谱

用量按 **2人份、牛肉280克、香芹300克**计算。做成偏川味的家常版本：**牛肉嫩滑、芹菜脆爽、微辣红亮，但豆瓣味不会盖住芹菜香**。

# 一、买菜时看

- **牛肉：** 250–300克，优先买牛里脊、嫩肩肉、吊龙或适合炒制的牛肉片；不要买牛腱、牛腩。让老板切片也可以，但要说明“横着纹路切薄片，炒牛肉用”。
- **芹菜：** 香芹约300克。选茎秆翠绿、挺直、水分足的，不要选粗老、空心、纤维特别明显的。
- **新鲜配料：** 姜1小块、蒜4瓣、泡椒3个。没有泡椒可换成小米辣2个；不想吃辣可以不放。
- **调味料：** 郫县豆瓣酱、生抽、蚝油、白胡椒粉、白糖、玉米淀粉、食用油。

最终比例约为 **牛肉1份、芹菜1份略多**。牛肉不要超过350克，否则普通家用炒锅容易降温，炒成出水的“煮牛肉”。

# 二、备菜时看

## 1. 切牛肉

观察牛肉纹路，刀口与纹路保持垂直，**逆着纹路切成2–3毫米薄片**。牛肉表面水分很多时，用厨房纸稍微吸干。

> 顺着纹路切，炒出来容易嚼不动；切得太厚，也很难在短时间内炒嫩。

## 2. 腌牛肉

牛肉放入大碗，依次加入：

**生抽1大勺、蚝油1小勺、白糖半小勺、白胡椒粉少许、清水1大勺。**

用手或筷子顺着一个方向抓拌 **1–2分钟**，直到水分被牛肉完全吸收；再加入 **玉米淀粉1.5小勺**抓匀，最后加入 **食用油1大勺**拌匀封住表面。

静置腌制 **15–20分钟**。

如果买到的牛肉明显偏老，可以在腌料里加 **八分之一小勺小苏打**，不要超过这个量，否则会有碱味。嫩牛里脊不需要加。

## 3. 处理芹菜

芹菜摘掉根部和老叶，嫩叶留下一小把。洗净后一定要 **充分沥干水分**。

芹菜秆切成 **4–5厘米斜段**；比较粗的芹菜秆先纵向剖成两半再切。不要提前放盐，否则炒之前就会出水。

## 4. 提前分好三份料

**香料碗：** 姜5片、蒜4瓣切片、泡椒3个切碎，全部放进同一个碗。

**酱汁碗：** 生抽1小勺、蚝油1小勺、白糖半小勺、清水1大勺，混合均匀。

**豆瓣小碟：** 郫县豆瓣酱1小勺。只放小勺，不要放一大勺，否则容易太咸，也会盖住芹菜和牛肉味。

把腌好的牛肉、芹菜、香料碗、酱汁碗和豆瓣小碟全部放到灶台旁边，正式炒的时候不要再找调料。

# 三、炒菜时手忙脚乱只看这里

## 1. 大火滑炒牛肉

炒锅烧干，开 **大火**烧到锅面明显发热，最好刚开始微微冒烟。

加入 **1.5大勺食用油**，转动锅身让油铺开，立即倒入全部牛肉，迅速摊开。

保持 **大火**，先不要反复翻动，煎约 **20–30秒**；底面变色后快速翻炒 **30–40秒**。看到大约七八成牛肉变色、还有少量粉红时，**马上盛出**。

> 牛肉此时没有完全熟是正常的，后面还要回锅。第一遍全部炒熟，第二遍就会变老。

如果锅比较小或牛肉超过300克，分两次炒，不要把牛肉堆成一团。

## 2. 小火炒豆瓣酱

锅里留下底油；油太少就补 **1小勺食用油**。

转 **小火**，加入豆瓣小碟里的 **郫县豆瓣酱1小勺**，翻炒 **约20秒**，看到红油出来即可。

不要长时间炒，豆瓣酱量少，很容易焦黑发苦。

## 3. 加香料碗

倒入准备好的 **香料碗**。

转 **中火**，快速翻炒 **20–30秒**，炒出姜、蒜和泡椒香味。蒜片刚开始变黄就进入下一步，不要等蒜炒焦。

## 4. 大火炒芹菜

倒入全部 **芹菜秆**，立即转 **大火**。

快速翻炒 **60–90秒**，让芹菜表面沾上红油并稍微变软，但仍然保持翠绿和脆度。

如果用的是较粗的西芹，可炒 **约2分钟**；细香芹不要炒太久。芹菜沥干后直接炒，锅里不要另外加水。

## 5. 牛肉立刻回锅

看到芹菜颜色变得更鲜亮时，**马上倒回刚才盛出的全部牛肉**。

保持 **大火**，快速翻炒 **约20秒**。

## 6. 加酱汁碗

沿锅边倒入准备好的 **酱汁碗**。

保持 **大火**，快速翻炒 **30–40秒**，让酱汁均匀包裹牛肉和芹菜。

加入留下的芹菜嫩叶，再翻炒 **约10秒**，牛肉完全变色后立即关火。

先尝一片芹菜再决定是否补盐。豆瓣酱、生抽和蚝油本身都有盐味，通常不需要另外加盐。

# 成品判断

- **牛肉：** 完全变色，表面有薄薄汁水，夹起来柔软，不干、不柴。
- **芹菜：** 颜色翠绿，表面入味，咬下去仍有明显脆感。
- **锅底：** 只有少量浓汁，没有一大摊水。
- **整体：** 微辣、咸鲜、略带泡椒香，颜色红亮但不是满盘豆瓣酱。

## 最简操作口令

**逆纹切牛肉 → 生抽蚝油清水抓吸收 → 加淀粉、加油腌15分钟 → 大火滑牛肉至八成熟盛出 → 小火炒豆瓣 → 中火下香料碗 → 大火炒芹菜60–90秒 → 牛肉回锅 → 倒酱汁碗炒30秒 → 加芹菜叶，立刻关火。**
