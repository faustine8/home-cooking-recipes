# 红汤萝卜烧牛肉

用量按 **2人份、牛腩或牛肋条600–700克、白萝卜约600克、燃气灶高压锅**计算。文中的“大勺”约15毫升，“小勺”约5毫升。本版用少量火锅底料增加红汤的厚味，但不会做成明显的火锅味。

# 一、买菜时看

* **主料：** 牛腩或牛肋条600–700克；中等大小白萝卜1根，约600克。
* **新鲜配料：** 姜1块、蒜1头、大葱1根；香菜或小葱少许，可不买。
* **香料：** 干辣椒2–4个、花椒半小勺、八角1个、香叶2片、桂皮1小段。
* **调味料：** 郫县豆瓣酱、生抽、老抽、蚝油、冰糖、盐。
* **千万别忘：** 牛油火锅底料一小块，约 **20克**。

最终口味是 **红汤浓香、微辣酱香、牛肉软烂、萝卜吸满汤汁**。火锅底料不要超过25克，否则容易盖住牛肉和萝卜的味道，变成火锅汤。

# 二、备菜时看

## 1. 焯牛肉

牛肉切成 **3–4厘米大块**，切得太小，高压后容易缩小、碎散。

锅中放牛肉，加足量冷水，再放 **姜4片、大葱半根**。

**大火煮开，撇掉浮沫；保持中大火再煮4分钟。** 捞出牛肉，用温水冲净表面的血沫，彻底沥干。

本版不需要料酒，姜和大葱已经足够处理腥味。

## 2. 处理萝卜

白萝卜削皮，切成 **4厘米左右滚刀块**。萝卜单独放好，暂时不要加入牛肉，也不要和牛肉一起高压。

## 3. 提前分好四份料

**香料碗：** 姜6片、蒜6瓣、干辣椒2–4个、花椒半小勺、八角1个、香叶2片、桂皮1小段，全部装进同一个碗。

**酱汁碗：** 生抽2大勺、老抽半大勺、蚝油1大勺、冰糖5–8颗，放进同一个碗。

**豆瓣小碟：** 郫县豆瓣酱1大勺，约15克。

**火锅底料小碟：** 火锅底料约20克，单独装碟，直接摆在高压锅旁边，避免忘记。

另外烧一壶热水，准备约 **700–900毫升**。想要红汤多一些就准备900毫升，但高压锅内食材和汤水不要超过安全刻度。

# 三、炒菜时手忙脚乱只看这里

## 1. 炒香料碗

炒锅烧干，放 **1.5大勺食用油**，倒入准备好的 **香料碗**。

保持 **中火** 翻炒约 **45秒**，炒出姜蒜和香料的味道。蒜表面略微发黄即可，不要把干辣椒和香料炒黑。

## 2. 加豆瓣酱

把香料拨到锅边，锅中央放入 **豆瓣小碟里的郫县豆瓣酱**。

立即转 **小火**，单独炒 **30–45秒**，看到豆瓣酱析出红油后，再和香料混合。

> 这里只炒豆瓣酱，不放火锅底料。火锅底料留到加水炖煮时再放。

## 3. 立刻加入牛肉

倒入全部焯好并沥干的牛肉。

转 **中大火**，持续翻炒 **2–3分钟**，让牛肉表面裹上豆瓣红油，并炒掉表面的水汽。

## 4. 加酱汁碗

倒入准备好的 **酱汁碗**。

保持 **中大火**，翻炒约 **1分钟**，让生抽、老抽、蚝油和冰糖均匀裹住牛肉。

## 5. 转入高压锅，加火锅底料

把炒锅里的牛肉、香料和全部酱汁倒入高压锅。

倒入热水，水面以 **刚好没过牛肉，最多再高出1厘米**为准。红汤想多一些可以适量增加，但不要一次倒得过满。

现在加入准备好的 **火锅底料小碟，约20克**，翻动几下，让底料浸入热汤中。

> **固定记忆：豆瓣酱炒，火锅底料炖。**

此时先不要放萝卜，也不要放盐。

## 6. 高压炖牛肉

盖好高压锅。

先开 **大火**，等高压锅稳定上汽后，立即转 **小火**，保持压力阀稳定工作即可，不要持续剧烈喷气。

从稳定上汽开始计时：

* 普通牛腩、牛肋条：压 **25分钟**；
* 筋膜较多、喜欢更软烂：压 **30分钟**。

时间到后关火，静置 **12–15分钟自然泄压**。不要立即强行放气，避免滚烫的油汤喷出。

## 7. 开锅检查牛肉

确认完全泄压后开盖，夹一块牛肉检查。

用筷子能比较轻松扎入，但牛肉仍能完整夹起来，就是合适。

如果牛肉仍然明显偏硬：先不要放萝卜，重新盖锅，大火上汽后转小火，再压 **5分钟**，继续自然泄压。

## 8. 放萝卜煮入味

确认牛肉已经软烂后，把白萝卜全部倒入高压锅。

这一步 **不要再盖高压锅盖加压**。

先开 **中大火煮开**，然后转 **中小火**，保持汤汁轻微沸腾，开盖煮 **15–20分钟**。中途翻动1–2次，让上层萝卜也能浸入红汤。

萝卜能被筷子较轻松扎透，但仍保持完整形状时即可。

## 9. 调味并调整红汤

先尝汤汁，再决定是否加盐：

* 淡：补少量盐，每次最多半小勺；
* 不够鲜：补半大勺生抽；
* 颜色偏浅：补少量老抽，每次最多半小勺；
* 太咸或太浓：加少量热水重新煮开；
* 火锅味太明显：加热水稀释，不要再加豆瓣酱。

如果汤汁过多，开 **大火收3–5分钟**；如果准备拌饭，不要收得太干，锅底保留 **约一大碗红汤**。

关火后夹出八角、桂皮和香叶，撒上小葱或香菜即可。

# 最简操作口令

**冷水焯牛肉 → 中火炒香料碗 → 小火炒豆瓣红油 → 中大火下牛肉 → 中大火倒酱汁碗 → 转入高压锅，加热水和火锅底料 → 大火上汽、小火压25–30分钟 → 自然泄压 → 放萝卜，中小火开盖煮15–20分钟 → 尝味调盐，保留红汤。**

# 最容易忘的三个重点

1. **豆瓣酱小火炒，火锅底料加水后再炖。**
2. **萝卜必须等牛肉压软后再放，不能和牛肉一起高压。**
3. **盐最后尝过汤再加，豆瓣酱、火锅底料和生抽本身都有咸味。**
